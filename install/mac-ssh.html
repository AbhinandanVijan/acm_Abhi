If you don’t already have one, make a GitHub account, you’ll need to write the same email address as your github account in the indicated placeholder location. Unless otherwise indicated, you’re typing all of these commands directly into the terminal.

1.
ssh-keygen -t ed25519 -C "replace-this@placeholder-example.com"

2. For the next three prompts it asks you, you can just hit enter.

3. eval "$(ssh-agent -s)"

4. Double check that you don’t already have an ssh config file using: open ~/.ssh/config

You probably don’t have one, and if it says you don’t: touch ~/.ssh/config then, open ~/.ssh/config

5. Copy and paste the following into the config file you just opened, then save using Command+S, and close out the file. Host github.com 
	AddKeysToAgent yes
	IdentityFile ~/.ssh/id_ed25519

6. ssh-add --apple-use-keychain ~/.ssh/id_ed25519

7. pbcopy < ~/.ssh/id_ed25519.pub

8. Go to github.com, click on your account, go to settings, click on “SSH and GPG keys”, then click “New SSH Key”, then do Command+F to paste in the key field (the last command copied the public key you need.) Name the key whatever you want, then click “Add SSH Key” and you’re done!
