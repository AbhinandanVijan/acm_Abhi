Make sure you don't already have git installed by entering "git" in the terminal. If a bunch of commands pop up, it's installed.

Installing git on mac should be pretty simple. It can easily go wrong though, so if you run into issues, ask one of us for some help! Enter these commands in the terminal:

1.
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2.
brew install git

That's it!
