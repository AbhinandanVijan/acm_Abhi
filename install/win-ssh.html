If you don’t already have one, make a GitHub account, you’ll need to write the same email address as your github account in the indicated placeholder location.

SSH Setup on windows requires entering commands in three different apps, Git Bash, PowerShell (start it as an administrator by right clicking it's icon and selecting "run as administrator"), and a terminal window. In which window to write commands will be indicated.

You can paste in PowerShell and Terminal by right clicking!

1. In Git:
ssh-keygen -t ed25519 -C "replace-this@placeholder-example.com"

2. In Git: For the next three prompts it asks you, you can just hit enter.

3. In PowerShell: Get-Service -Name ssh-agent | Set-Service -StartupType Manual
Start-Service ssh-agent

5. In Terminal: ssh-add c:/Users/YOU/.ssh/id_ed25519

6. In Terminal: clip < ~/.ssh/id_ed25519.pub

If the prior command throws an error on your system, try: cat ~/.ssh/id_ed25519.pub | clip

8. Go to github.com, click on your account, go to settings, click on “SSH and GPG keys”, then click “New SSH Key”, then do Command+F to paste in the key field (the last command copied the public key you need.) Name the key whatever you want, then click “Add SSH Key” and you’re done!

If your system says it can't verify github's authenticity, just enter "yes" as a response
